Mini projet : Découvrir Git

Un petit projet avec trois fichiers (HTML, CSS, JS) pour comprendre les bases de Git.

1. Objectif du projet

Ce projet sert d’exemple minimal pour apprendre à utiliser Git :

  • voir l’état des fichiers avec git status
  • préparer les changements avec git add
  • créer des versions avec git commit

Le but est de faire plusieurs petits changements dans ce projet et de les enregistrer dans l’historique avec des commits.

2. Exercice : Comprendre les commandes Git

Avant de cliquer sur les boutons, essayez d’expliquer vous-même à quoi sert chaque commande. Ensuite, appuyez sur « Afficher le rôle » pour vérifier.

  • git init

  • git status

  • git add <fichier>

  • git commit -m "message"

  • git log

Mini projet Git – 3 fichiers : index.html, css/style.css, js/main.js.
